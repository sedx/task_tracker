== Нстройка

Выполните следующие шаги
* Склонируйте репозиторий

* Настройте подключение к БД

* Создате БД и выполните миграции

* Запустите сервер

* Зарегистрируйтесь

* Выполните `rake admin:crate_admin`

== Реализовано
* Авторизация и Аутентификация пользователей
* Создание проектов админом
* Назначение пользователей на проект
* Создание, Редактирвоание и удаление таски
* Изменение статсуа таски перетаскиванием
